# ALX Travel App 0x00

A simple Django project for managing travel listings, bookings, and reviews.  
This project demonstrates **database modeling, serializers for API representation, and seeding the database with sample data**.

## Features

- **Listings**: Create and manage travel listings with details like title, description, price, and location.
- **Bookings**: Users can book listings for specific dates with automatic total price calculation.
- **Reviews**: Users can leave reviews and ratings for listings.
- **API-ready**: Models have serializers to easily expose data through Django REST Framework.
- **Database Seeder**: Populate the database with sample listings using a custom management command.
